範圍練（scope chain）

範圍練（scope chain）是立即執行函式（Immediately Invoked Function Expression）與閉包的共同重要觀念，
Javascript function 的靜態物理坐落位置（physical position）深深影響著外部環境（outer environments），
也取決了那些變數是可以取用（accessible）與共享的。
